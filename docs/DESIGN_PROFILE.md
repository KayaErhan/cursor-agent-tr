# Spor Tesisi Uygulamasi - Tasarim Profili

## Tasarim Profili

- Profil: **kurumsal**
- Tema modu: **hepsi** (dark, soft-dark, light arasi switcher)

## Admin Panel Layout

- **Sol Sidebar:**
  - Logo ve tesis adi
  - Ana menü basliklari:
    - Dashboard
    - Rezervasyonlar
    - Salon/Saha Takvimi
    - Uyeler
    - Antrenorler
    - Paketler ve Ucretlendirme
    - Raporlar
    - Ayarlar

- **Topbar:**
  - Sag ustte:
    - Bildirim ikonu (rezervasyon onayi, iptal, odeme hatalari)
    - Email/mesaj ikonu (uye ve antrenor bildirimleri)
    - Kullanici profil menusu (yonetici bilgileri, cikis vb.)
  - Tema switcher:
    - Dark / Soft-Dark / Light secenekleri arasinda gecis.

## Dashboard Bilesenleri

- KPI Kartlari:
  - Bugunku/toplam rezervasyon sayisi
  - Doluluk oranlari (gunluk/haftalik)
  - Aylik gelir ozetleri
  - Iptal oranlari

- Grafikler:
  - Rezervasyon zaman dagilimi (line chart)
  - Gelir dagilimi (bar chart)
  - Paket/uyelik turu dagilimi (donut/pie chart)

- Son Aktiviteler:
  - Son yapilan rezervasyonlar
  - Iptaller
  - Yeni uye kayitlari

## UI Teknolojileri

- Tailwind CSS (zorunlu)
- Ikonlar icin `lucide-react`
- Grafikler icin `recharts`
- Mikro animasyonlar icin `framer-motion`

Bu profil, admin panel ve olasi landing/site tarafinda kurumsal, renkli ve modern bir gorunum saglayacak sekilde kullanilmalidir.
